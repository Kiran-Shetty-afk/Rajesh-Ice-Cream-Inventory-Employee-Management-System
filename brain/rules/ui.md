# UI and design

## Baseline

- Preserve the existing quiet dashboard/admin style.
- Use compact, scannable tables and clear action controls.
- Prefer lucide-react icons already used in the project.
- Keep forms accessible with labels, helpful validation messages, and visible focus.

## Layout

- The app uses a fixed left sidebar and main content area.
- Existing pages use rounded-md bordered white sections on a warm light background.
- Avoid landing-page patterns; this is an operational desktop tool.

## Future UI work

- Editable workflows should feel fast for daily owner/admin use.
- Search and filters should be near the data they control.
- Reports should be print-friendly and include business name, date, filters, and totals.
